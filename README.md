<h1>Little Lexikon</h1>

Little Lexikon is a compact lexicon system built with C# and Microsoft SQL. It provides a simple internal database for managing terms and short definitions.

<h2>Features</h2>

Store and manage terms with associated definitions
Add, edit, and remove entries


<h2>Usage</h2>

The application allows creating, reading, updating, and deleting lexicon entries and can be expanded to fit various knowledge-management use cases.
